## スクリプト使用方法

1. 5chのURLを取得する
1. 以下のようにコード実行する
    ```shell
    python get_5ch.py https://asahi.5ch.net/newsplus/dat/1723624289.dat [--mode 0]
    ```

    run.batも参考になる
